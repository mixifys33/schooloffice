🔧 Staff Onboarding Button Test Guide

📍 Where to Find the Button

The staff onboarding button is located in the top header/navigation bar of your dashboard, next to the theme toggle and notifications. Look for:

  • A UserCog icon (⚙️👤)
  • It appears only for School Admin users
  • It may have different visual states based on your setup status

🎯 Button Visual States

The onboarding button will appear in different states depending on your school's setup:

1. Normal State (Setup Complete)

Gray/neutral appearance when all required staff are registered

2. Attention State (Setup Incomplete)

Amber/yellow background with red notification dot when staff setup is needed

3. Loading State

Shows spinner when clicked and loading

🔍 How to Test

  1. Login as School Admin - The button only appears for this role
  2. Look in the header - Top right area, near theme toggle
  3. Check for visual indicators:
    • Red dot = Setup incomplete
    • Amber background = Action needed
    • Spinner = Loading after click
  4. Click the button - Should show loading animation then open modal

🐛 Debugging Steps

If you can't see the button, run this in your browser console:

// Check if you're logged in as School Admin fetch('/api/auth/session') .then(r => r.json()) .then(session => { console.log('User role:', session?.user?.role); console.log('Should see button:', session?.user?.role === 'SCHOOL_ADMIN'); }); // Look for the button in DOM const button = document.querySelector('button[title*="Staff Onboarding"]'); console.log('Button found:', !!button); if (button) { console.log('Button element:', button); button.style.border = '3px solid red'; // Highlight it } // Check onboarding status fetch('/api/staff/onboarding/status') .then(r => r.json()) .then(data => { console.log('Onboarding status:', data); console.log('Setup complete:', data.isComplete); console.log('Missing roles:', data.missingRoles); });

✅ Expected Behavior

  • Button visible - Only for School Admin users
  • Visual feedback - Different states based on setup status
  • Loading animation - Spinner appears when clicked
  • Modal opens - Staff onboarding dialog appears
  • Dashboard banner - Prominent banner on dashboard if setup incomplete

🚨 Still Not Working?

If the button still doesn't appear or work:

  1. Refresh the page completely
  2. Check browser console for errors
  3. Verify you're logged in as SCHOOL_ADMIN
  4. Run the debugging script above
  5. Look for the prominent banner on the dashboard page
