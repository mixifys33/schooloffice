# SchoolOffice

SchoolOffice is a comprehensive school management system built for small to medium-sized schools in Uganda.

## Core Features
- Student and staff management
- Fees, invoices, receipts, and financial reports
- Attendance and academic records
- Parent, student, and teacher portals
- SMS and communication automation
- Role-based access control and audit logs

## Tech Stack
- Next.js (App Router)
- TypeScript
- Prisma
- PostgreSQL
- SMS and email integrations

## Status
Active development. Early pilot schools onboarding.
